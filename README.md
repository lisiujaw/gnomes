# Gnome manager
Manage your virtual gnomes.

## Instalation
1. Git clone :)
2. composer install
3. Copy .env.example file to .env and modify it
4. Run php artisan key:generate
5. Run php artisan migrate --seed
6. You`r ready to manage gnomes! Have fun!

## Requirements
* PHP7
* MySQL
* Installed PHP GD Library
